# Evidence Notes

## What This Repository Is Meant to Prove

This repository is intended to show that the project is a runnable prototype rather than a static project description.

## Verification Points

1. `streamlit_app.py` contains a real API call path
2. The UI exposes:
   - API key input
   - base URL
   - model name
   - generated output
3. The app displays validation details:
   - latency
   - input length
   - output length
